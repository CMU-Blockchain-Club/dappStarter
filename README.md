This project was created to help Carnegie Mellon Blockchain students build their first dapp. The code is well documented and provides a generic interface for connecting web3 wallets.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.tsx`. The page auto-updates as you edit the file.
